Hugo 实现深浅主题的切换

2023-02-02
3 minute read
Web
hugo , css , js

实现深浅主题切换的方法不只一种，这里记录我所使用的一种。

生成深浅主题对应的 css 文件

配置入口文件

我的博客通过 sass 构建样式，可以将写好的 scss 文件组合生成一个 css 文件。

而组合成为 css 文件需要一个入口文件，在入口文件里指定主题。

比如我在 assets/scss/colors 中指定了两种色调 dark.scss 和 light.scss, 那么可以在对应的入口文件中 import:

1// assets/scss/styles_dark.scss
2@import "colors/dark";
3
4@import "variables";
5@import "util";
6@import "mixins";
7// ...

1// assets/scss/styles_light.scss
2@import "colors/light";
3
4@import "variables";
5@import "util";
6@import "mixins";
7// ...

通过 hugo 生成目标文件

接着，在 <head></head> 中指定入口文件进行操作：

1{{ $opts := (dict "targetPath" "css/styles_dark.css" "outputStyle" "compressed") }}
2{{ $styles_dark := resources.Get "scss/styles_dark.scss" | resources.ExecuteAsTemplate "scss/styles_dark.scss" . | resources.ToCSS $opts }}
3
4{{ $opts := (dict "targetPath" "css/styles_light.css" "outputStyle" "compressed") }}
5{{ $styles_light := resources.Get "scss/styles_light.scss" | resources.ExecuteAsTemplate "scss/styles_light.scss" . | resources.ToCSS $opts }}
6
7{{ $dark := $styles_dark.Permalink }}
8{{ $light := $styles_light.Permalink }}

这里通过 hugo 操作之后，即可生成深浅主题对应的两个 css 文件，且将其地址赋值到 $dark 和 $light 中。

保存地址变量

接着，通过隐藏域保存这两个地址以便后续使用：

1<input value="{{ $dark }}" id="styles-dark" style="display: none;" />
2<input value="{{ $light }}" id="styles-light" style="display: none;" />

设定初始主题

一开始的主题应由网站发布者指定，读取 config 文件中的变量即可：

1{{ $dark := $styles_dark.Permalink }}
2{{ $light := $styles_light.Permalink }}
3{{ if eq .Site.Params.colortheme "dark" }}
4<input value="dark" id="theme-current" style="display: none;" />
5<link id="theme-css" rel="stylesheet" href="{{ $dark }}" />
6{{ else if eq .Site.Params.colortheme "light" }}
7<input value="light" id="theme-current" style="display: none;" />
8<link id="theme-css" rel="stylesheet" href="{{ $light }}" />
9{{ end }}

这里还配置了隐藏域，便于后续确认和修改主题。

通过 js 动态设置主题

加载主题

步骤如下：

  • 读取用户 localStorage 中保存的主题变量
  • 读取当前主题隐藏域
  • 读取 css 文件地址隐藏域中的文件地址
  • 修改 css 标签中的地址以加载主题
 1function setTheme(theme) {
 2  let stylesDark = document.getElementById("styles-dark").value;
 3  let stylesLight = document.getElementById("styles-light").value;
 4  let themeCss = document.getElementById("theme-css");
 5  let setThemeStyle = theme === "dark" ? stylesDark : stylesLight;
 6  themeCss.setAttribute("href", setThemeStyle);
 7}
 8
 9function getCurrentTheme() {
10  // user theme
11  let userTheme = localStorage.getItem("userTheme");
12  let userThemeExpire = localStorage.getItem("userThemeExpire");
13  if (
14    userTheme !== null &&
15    userTheme != "" &&
16    userThemeExpire !== null &&
17    userThemeExpire > new Date().getTime()
18  ) {
19    document.getElementById("theme-current").setAttribute("value", userTheme);
20  }
21  return document.getElementById("theme-current").value;
22}
23
24setTheme(getCurrentTheme());

修改主题

每次修改主题都需要修改当前主题隐藏域中的值，并且在用户 localStorage 中存储用户期望的主题，设置超时时间则是一个可选项：

1function changeTheme(theme) {
2  setTheme(theme);
3  document.getElementById("theme-current").setAttribute("value", theme);
4  localStorage.setItem("userTheme", theme);
5  // 设置超时时间： 24 小时
6  let expire = new Date().getTime() + 1000 * 3600 * 24;
7  localStorage.setItem("userThemeExpire", expire);
8}
