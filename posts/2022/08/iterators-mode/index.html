迭代器模式

2022-08-25
4 minute read
Language
design , java

定义

迭代器模式提供一种方法顺序访问一个聚合对象中的各个元素，而又不暴露其内部的表示。

通过一个Iterator，可以方便地遍历各种类型，如HashMap，ArrayList等。

代码展示

下面通过菜单的例子进行说明。

分别用简单数组String[]和ArrayList<String>类创建菜单内容。

从接口开始

创建菜单接口：

1public interface Menu {
2	public Iterator<String> createIterator();
3}

每个菜单都可以返回一个迭代器。

创建Diner菜单

该菜单用String[]创建内容，需要编写自己的迭代器。

 1public class DinerMenu implements Menu {
 2	static final int MAX_ITEMS = 6;
 3	int numberOfItems = 0;
 4	String[] menuItems;
 5  
 6	public DinerMenu() {
 7		menuItems = new String[MAX_ITEMS];
 8 
 9		addItem("Vegetarian BLT");
10		addItem("BLT");
11		addItem("Soup of the day");
12		addItem("Hotdog");
13		addItem("Steamed Veggies and Brown Rice");
14		addItem("Pasta");
15	}
16  
17	public void addItem(String name) 
18	{
19		if (numberOfItems >= MAX_ITEMS) {
20			System.err.println("Sorry, menu is full!  Can't add item to menu");
21		} else {
22			menuItems[numberOfItems] = name;
23			numberOfItems = numberOfItems + 1;
24		}
25	}
26 
27	public String[] getMenuItems() {
28		return menuItems;
29	}
30  
31	public Iterator<String> createIterator() {
32		return new DinerMenuIterator(menuItems);
33	}
34 
35	public String toString() {
36		return "Diner Menu";
37	}
38	// other menu methods here
39}

创建菜单迭代器

实现Iterater<String>接口，实现next()，hasNext()和remove()方法。

 1public class DinerMenuIterator implements Iterator<String> {
 2	String[] list;
 3	int position = 0;
 4 
 5	public DinerMenuIterator(String[] list) {
 6		this.list = list;
 7	}
 8 
 9	public String next() {
10		String menuItem = list[position];
11		position = position + 1;
12		return menuItem;
13	}
14 
15	public boolean hasNext() {
16		if (position >= list.length || list[position] == null) {
17			return false;
18		} else {
19			return true;
20		}
21	}
22  
23	public void remove() {
24		if (position <= 0) {
25			throw new IllegalStateException
26				("You can't remove an item until you've done at least one next()");
27		}
28		if (list[position-1] != null) {
29			for (int i = position-1; i < (list.length-1); i++) {
30				list[i] = list[i+1];
31			}
32			list[list.length-1] = null;
33		}
34	}
35}

测试

printMenu方法通过一个迭代器遍历并打印菜单。

 1public class Cafe {
 2
 3	public static void main(String args[]) {
 4		PancakeHouseMenu pancakeHouseMenu = new PancakeHouseMenu();
 5		DinerMenu dinerMenu = new DinerMenu();
 6		
 7		// with iterators
 8		Iterator<String> pancakeIterator = pancakeHouseMenu.createIterator();
 9		Iterator<String>  dinerIterator = dinerMenu.createIterator();
10
11		System.out.println("\nMENU (with iterators)\n----\nBREAKFAST");
12		printMenu(pancakeIterator);
13		System.out.println("\nLUNCH");
14		printMenu(dinerIterator); 
15	}
16 
17	private static void printMenu(Iterator<String> iterator) {
18		while (iterator.hasNext()) {
19			String menuItem = (String)iterator.next();
20			System.out.println(menuItem);
21
22		}
23	}
24}
