Quartz 实现定时任务

2022-06-15
2 minute read
Language
java

一、添加注解

主启动类添加@EnableScheduling注解；

二、添加依赖：

1<!--定时任务 -->
2<dependency>
3    <groupId>org.springframework.boot</groupId>
4    <artifactId>spring-boot-starter-quartz</artifactId>
5</dependency>

三、创建任务

/TestTask1.java

1public class TestTask1 extends QuartzJobBean {
2    @Override
3    protected void executeInternal(JobExecutionContext context) {
4        SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
5        System.out.println("Test1---" + sdf.format(new Date()));
6    }
7}

/TestTask2.java

1public class TestTask2 extends QuartzJobBean {
2    @Override
3    protected void executeInternal(JobExecutionContext context) {
4        SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
5        System.out.println("Test2---" + sdf.format(new Date()));
6    }
7}

四、配置任务

/QuartzConfig.java

注意同组下同名任务不会重复执行。

  • JobDetail：配置任务信息，生成任务
  • Trigger：配置触发器信息，设置时间
 1@Configuration
 2public class QuartzConfig {
 3    @Bean
 4    public JobDetail testJ1() {
 5        return JobBuilder.newJob(TestTask1.class).withIdentity("job1", "group").storeDurably().build();
 6    }
 7    @Bean
 8    public JobDetail testJ2() {
 9        return JobBuilder.newJob(TestTask2.class).withIdentity("job2", "group").storeDurably().build();
10    } // "job2"如果为"job1"，则不会执行TestTask2而是执行TestTask1
11
12    @Bean
13    public Trigger test1() {
14        SimpleScheduleBuilder ssb = SimpleScheduleBuilder.simpleSchedule().withIntervalInSeconds(59).repeatForever();
15        return TriggerBuilder.newTrigger().forJob(testJ1())
16                .withIdentity("trigger1", "group")
17                .withSchedule(ssb)
18                .build();
19    }
20
21    @Bean
22    public Trigger test2() {
23        return TriggerBuilder.newTrigger().forJob(testJ2())
24                .withIdentity("trigger2", "group")
25                .withSchedule(CronScheduleBuilder.cronSchedule("0/59 * * * * ?"))
26                .build();
27    }
28}
