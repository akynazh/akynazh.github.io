查看内网IP

grep抓取inet行，sed删除inet字串和netmask.*字串

1alias myip=" ifconfig eth0 | grep 'inet ' | sed 's/inet//g' | sed 's/netmask.*$//g'"

test:

1[root@VM-0-11-centos ~]# myip
2         172.17.0.11

查看外网IP

1curl cip.cc

show:

IP : ${my_ip} 地址 : 中国 中国

数据二 : 上海市 | 腾讯云

数据三 : 中国北京北京 | 腾讯

URL : http://www.cip.cc/${my_ip}

查看IP端口状况

1[root@VM-0-11-centos ~]# netstat -ntlp
2Active Internet connections (only servers)
3Proto Recv-Q Send-Q Local Address           Foreign Address         State       PID/Program name
4tcp        0      0 0.0.0.0:22              0.0.0.0:*               LISTEN      1089/sshd
5
6tcp6       0      0 :::3306                 :::*                    LISTEN      14528/mysqld

参数说明：

  • n：拒绝别名都用数字
  • t：即只接受TCP
  • l：即只显示在listen中的
  • p：可显示相关程序名

另外一种方法：

获取3306端口运行状况：

1[root@VM-0-11-centos ~]# lsof -i:3306
2COMMAND   PID  USER   FD   TYPE DEVICE SIZE/OFF NODE NAME
3mysqld  14528 mysql   17u  IPv6 779103      0t0  TCP *:mysql (LISTEN)

开启与关闭端口

这里实验环境为centos7.9，我采用firewall来实现需求。

 1# 首先打开防火墙服务
 2[root@VM-0-11-centos ~]# systemctl start firewalld 
 3
 4# 永久开启80端口
 5[root@VM-0-11-centos ~]# firewall-cmd --zone=public --add-port=8080/tcp --permanent 
 6
 7# 使生效
 8[root@VM-0-11-centos ~]# firewall-cmd --reload
 9
10# 查看已经开启的端口
11[root@VM-0-11-centos ~]# firewall-cmd --zone=public --list-ports
128080/tcp 3306/tcp 80/tcp
