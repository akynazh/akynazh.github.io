Open, Connect And Config

  1. open redis:
1redis-server
  1. connect to redis:
1redis-cli
  1. disconnect:
1redis-cli shutdown
  1. show chinese:
1redis-cli --raw
  1. connect to remote redis:
1redis-cli -h host -p port -a password
  1. get redis config:
1config get {config_name}

Password的设置

 2127.0.0.1:6379> config get requirepass

 42) ""
 5127.0.0.1:6379> config set requirepass 12345
 6OK
 7127.0.0.1:6379> config get requirepass

 9127.0.0.1:6379> auth 12345 # 进行密码验证
10OK
11127.0.0.1:6379> config get requirepass
121) "requirepass"
132) "12345" # 发现已经有密码咯
14127.0.0.1:6379> exit
15PS C:\Users\akyna> redis-cli
16127.0.0.1:6379> auth 12345
17OK
18127.0.0.1:6379> config get requirepass
191) "requirepass"
202) "12345"

如果重启server密码会失效。

一般在Linux下主机重启后需要重新配置一下密码。

可以修改redis.conf文件，设置密码:

1requirepass ${password}

以后通过下面命令：

1redis-server redis.conf

进行服务重新启动时的配置，这样可以在服务开启时同时指定密码。
