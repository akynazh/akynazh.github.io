2021-12-05
3 minute
Language
cpp

关于char[]与char*的区别

1char s1[] = "abc";
2cout << s1 << endl; // abc
3s1[1] = 'd'; // 改变值
4cout << s1 << endl; // adc

“abc"为字符串常量，s1为指针常量，即指针是一个常量，所以指向地址不能改变，指向地址的值可变

1const char* s2 = "abc"; // const是默认存在的
2cout << s2 << endl; // abc
3s2 = "dfs"; // 改变地址
4cout << s2 << endl; // dfs

“abc"为字符串常量，s2为常量指针，指向地址可变，但指向的地址的值不能变

以上两者皆可用于函数传值

1void print_str(const char * str) {
2    printf("%s\n", str);
3}
4
5print_str(s1); // adc
6print_str(s2); // dfs

关于string和char[],char*的关系

  • string是类，string是对一个字符串的引用值
1const char* s1 = "abc";
2string s = s1;
3cout << s << endl; // abc
4s += "a";
5cout << s << endl; // abca 
6cout << s1 << endl; 
7// abc 证明string只是引用了s1
  • string引用字符串示例
1const char *s1 = "123";
2string str1 = s1; // 若要这样赋值s1必须加const
3cout << str1 << endl; // 123
4
5char s2[] = "234";
6string str2 = s2;
7cout << str2 << endl; // 234
  • string转char*
1const char* s1_1 = str1.c_str(); 
2// c_str 转 const char*
3cout << s1_1 << endl; // 123

char*,char[]与int的相互转换

1int v1 = atoi(s1);
2cout << v1 << endl; // 123
3v1 += 1;
4char str11[100]; 
5itoa(v1, str11, 10); // 10代表十进制
6cout << str11 << endl; // 124
7// 下面的会失败
8// const char* str111;
9// itoa(v1, str111, 10); // 失败

string与int的相互转换

1int val = 223;
2string str3 = to_string(val); 
3// C++11新特性 只适用于string (char[]和char*不可以)
4cout << str3 << endl; // 223
5val = stoi(str3, 0, 10); // 0为起始位置 10即十进制
6cout << val << endl; // 223
