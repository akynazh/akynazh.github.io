Window 实用命令记录之文件操作

2021-04-19
1 minute read
Language
windows , cmd , powershell

创建文件或文件夹

cmd下创建文件

1type nul > test.jpg

powershell下创建文件

1new-item test.jpg

创建文件夹

1mkdir test

重命名文件

注意不能指定到新的路径。

1PS D:\a\tmp\test> ren test.txt my.txt

移动文件

该命令也可用于重命名。

1PS D:\a\tmp\test> move test.jpg mydir

复制文件和文件夹

copy无法递归复制，而xcopy可以。

1PS D:\a\tmp\test\mydir> copy test.jpg test1.jpg

 1# mydir not empty
 2PS D:\a\tmp\test> copy mydir mydir1
 3PS D:\a\tmp\test> ls mydir1
 4# empty
 5
 6PS D:\a\tmp\test> xcopy mydir mydir2
 7目标 mydir2 是文件名
 8还是目录名
 9(F = 文件，D = 目录)? D
10mydir\test.jpg
11mydir\test1.jpg
12复制了 2 个文件

删除文件或文件夹

del

删除文件。powershell可以使用rm。

rd（rmdir）

删除空目录。

rd -R递归删除非空目录。
